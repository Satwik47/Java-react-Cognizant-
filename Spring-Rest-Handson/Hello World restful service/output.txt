================================================================================
ANNOTATION EXPLANATIONS
================================================================================
@RestController:
- Marks the class as a REST Controller
- Combines @Controller and @ResponseBody
- The returned string is sent directly as the HTTP response body

@GetMapping("/hello"):
- Maps the HTTP GET request for /hello to the sayHello() method

================================================================================
METHOD EXECUTION FLOW
================================================================================
@GetMapping("/hello")
public String sayHello() {

    LOGGER.info("START");        // Step 1: Log START

    String message = "Hello World!!";  // Step 2: Create String

    LOGGER.info("END");          // Step 3: Log END

    return message;              // Step 4: Return response
}

Execution order:
1. Request received
2. START log
3. Create String
4. END log
5. Return response
